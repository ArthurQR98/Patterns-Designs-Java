# Patterns Desings -Chain Of Responsability- with Java 🚀

El patrón Chain of Responsibility construye una cadena de objetos tal que si un objeto de la cadena no puede responder a la solicitud, puede transmitirla a su sucesor y así sucesivamente hasta que uno de los objetos de la cadena responde.

## Use Case

* Una cadena de objetos gestiona una solicitud según un orden que se define
dinámicamente.
* La forma en que una cadena de objetos gestiona una solicitud no tiene por qué
conocerse en sus cliente
